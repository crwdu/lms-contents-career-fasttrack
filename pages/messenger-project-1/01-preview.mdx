## 2. 준비하기

이번 프로젝트를 위해 필요한 몇 가지를 먼저 준비해보도록 하겠습니다.

<br/>
### 2-1. 프로젝트 파일 다운로드 받기

다음 경로에서 프로젝트 파일을 다운로드 받고, 압축을 해제한 후 여러분이 편한 위치로 이동시켜보세요.

<Callout emoji="🗂️">
[실시간 메신저 프로젝트 파일 다운로드 받기](/files/Vaco-Messenger.zip)
</Callout>

<br/>
### 2-2. 프로젝트 파일 열기

<br/>
1. 컴퓨터에서 Visual Studio Code 프로그램을 열어보세요.

<Callout emoji="📌">
Visual Studio Code가 아직 없으시다면, 지금 [다운로드](https://code.visualstudio.com/download) 받으세요.
</Callout>

<br/>
2. "폴더 열기"를 선택하고, 여러분이 다운로드 받은 Vaco-Messenger 폴더를 선택하세요.

![VS Code open folder](/images/posts/open_folder.png)

<br/>
3. 다음과 같이 올바른 폴더가 열렸는지 좌측 패널의 폴더 구조를 잘 확인해보세요.

![VS Code open folder](/images/posts/opened_folder.png)

<br/>
### 2-3. Live Server 설치하기

아래 주소에서 Visual Studio Code용으로 개발된 Live Server 확장 프로그램을 설치하세요.

<Callout emoji="📌">
이 곳에서 Live Server를 [다운로드](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 받으세요.
</Callout>

> Live Server는 우리가 작업한 웹 사이트를 조금 더 편리하게 확인할 수 있게 도와주는 도구입니다.

<br/>
### 2-4. Live Server 실행하기

우리가 작업할 프로젝트 파일을 열어놓은 Visual Studio Code에서 우측 하단의 **Go Live**라고 표기된 버튼을 클릭해보세요.

![VS Code Go Live](/images/posts/live_server.png)

`127.0.0.1:5500`이라는 주소의 인터넷 창이 열리고, 개발자 도구를 열어보시면 아래와 같은 오류가 발생하는 상태인 것을 확인하실 수 있습니다.

![messenger initial screen](/images/posts/initial_screen.png)

<br/>
### 2-5. Live Server 실행 중지하기

작업하는 시간 동안 Live Server를 사용한 후, 작업이 끝나면 아래와 같이 **Port: 5500**이라고 표기된 버튼을 클릭하여 Live Server를 종료시켜주세요.

나중에 다시 작업을 재개하는 경우, Live Server를 다시 켜고 작업하세요.

![VS Code Turn Off](/images/posts/live_server_off.png)

<br/>
## 3. 현재 상태 살펴보기

Live Server를 켠 상태에서 사이트에 접속해보시면, 아래와 같은 화면이 나타납니다.

- 닉네임을 입력하고 키보드의 Enter 키를 입력해보세요. 화면에서 이름이 사라지게 됩니다.
- 닉네임을 입력하고 입장하는 버튼을 클릭해보세요. 화면 상으로 아무런 변화가 없습니다.

이와 같이 현재는 사용자가 채팅 화면으로 입장하는 기능이 정상적으로 동작하지 않고 있지만, 한 가지씩 고쳐가며 기능을 정상화해보도록 하겠습니다. 🤓
