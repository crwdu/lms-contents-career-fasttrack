## 다시 보는 자바스크립트 기초상식

### 자료형

자바스크립트에는 총 8가지 자료형이 있습니다.

- String
- Boolean
- Number
- Null
- Undefined
- Object (함수, 배열 포함)
- Symbol
- BigInt

`Symbol`과 `BigInt`를 제외하고, 이 정도 목록은 자바스크립트를 다루는 사람이라면 쿡 찔러서 10초 안에 술술 나올 수 있어야 합니다. 이번주 내에 반드시 외우도록 하세요!

### `&&`, `||` 연산자

생각보다 많은 분들이 논리 연산자에 대해 잘못 알고 있는 경우가 많습니다. 아래 내용을 꼼꼼히 읽어보세요.

자바스크립트의 모든 "값"들은 두 가지 부류로 나눌 수 있습니다. 바로, Truthy와 Falsy. 진실스러움이냐, 아니면 거짓스러움이냐 라는 의미입니다.

<br />
<br />

#### 거짓스러운 값들 (Falsy)

아래 나열된 값들이 모두 Falsy 값들입니다. 자바스크립트에서 Falsy 값들은 아래 나열된 값들이 유일합니다.

```js showLineNumbers copy
0
-0
false
undefined
null
""
''
``
NaN
```

<br />
<br />

#### 왜 Falsy, Truthy를 구분해야 할까요?

##### 예시 1.

```js showLineNumbers copy
if (0) {
  console.log("hello world");
}
```

위의 예시 코드에서 조건문의 조건은 충족하지 않습니다. 결과적으로 콘솔 출력문 또한 실행되지 않습니다. `0`이 Falsy 값이고, 자바스크립트에서 조건문은 Falsy, Truthy에 따라 조건문이 충족하는지 판단하기 때문입니다.

<br />
<br />

##### 예시 2.

```js showLineNumbers copy
for (let i = 0; i ; i++) {
  console.log(i);
}
```

위의 반복문 또한 전혀 반복하지 않고 종료됩니다. 왜냐하면 반복문이 반복하기 위해서는 소괄호 내부의 가운데 구절이 충족되어야 하는데, Truthy일 경우에만 반복문 내부로 진입하기 때문입니다. 현재 `i`의 값은 0이기 때문에 Falsy이고, 그렇기에 반복문 내부로 진입하지 않고 즉시 코드의 실행이 종료됩니다.

이제 정말 논리 연산자를 살펴보도록 할까요?

<br />
<br />

#### `&&` 연산자

<br />

<Alert
  showIcon
  message="&& 연산자는 좌측부터 시작하여 Falsy 값을 찾기 시작하고, Falsy 값을 찾을 경우 연산을 중지하고 해당 값을 결과값으로 사용합니다. 마지막까지 Falsy 값을 찾지 못한 경우, 마지막에 위치한 값이 최종 결과값이 됩니다."
  type="info"
/>

<br />
<br />

##### 예시 3.

```js showLineNumbers copy
const a = true && true;
```

1. 좌측부터 시작하여 Falsy 값을 찾습니다.
2. 좌측의 `true`는 Falsy가 아니므로, 다음 값을 판단합니다.
3. 우측의 `true` 또한 Falsy가 아니지만, 마지막 값이므로 마지막 값인 `true`를 결과값으로 사용합니다.
4. `a` 변수에 `true`가 담기게 됩니다.

<br />
<br />

##### 예시 4.

```js showLineNumbers copy
const a = true && false;
```

1. 좌측부터 시작하여 Falsy 값을 찾습니다.
2. 좌측의 `true`는 Falsy가 아니므로, 다음 값을 판단합니다.
3. 우측의 `false`는 Falsy이므로 결과값으로 사용합니다.
4. `a` 변수에 `false`가 담기게 됩니다.

<br />
<br />

##### 예시 5.

```js showLineNumbers copy
const a = "hello" && {};
```

1. 좌측부터 시작하여 Falsy 값을 찾습니다.
2. 좌측의 `"hello"`는 Falsy가 아니므로, 다음 값을 판단합니다.
3. 우측의 `{}`는 Falsy가 아니지만, 마지막 값이므로 결과값으로 사용합니다.
4. `a` 변수에 `{}`가 담기게 됩니다.

<br />
<br />

##### 예시 6.

```js showLineNumbers copy
const a = "" && "hello";
```

1. 좌측부터 시작하여 Falsy 값을 찾습니다.
2. 좌측의 `""`는 Falsy 값이 맞으므로 결과값으로 사용하고 연산이 종료됩니다.
3. `a` 변수에 `""`가 담기게 됩니다.

<br />
<br />

##### 예시 7.

```js showLineNumbers copy
const a = (function foo () {}) && "hello";
```

1. 좌측부터 시작하여 Falsy 값을 찾습니다.
2. 좌측의 `(function foo () {})`는 Falsy 값이 아니므로, 다음 값을 판단합니다.
3. 우측의 `{}`는 Falsy가 아니지만, 마지막 값이므로 결과값으로 사용합니다.
4. `a` 변수에 `"hello"`가 담기게 됩니다.

<br />
<br />
