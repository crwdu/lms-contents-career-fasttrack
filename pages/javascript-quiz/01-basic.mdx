# 문법 기초 퀴즈

아래 코드 실행문들이 어떤 결과를 출력하게 될지 고민해보고, 코드를 실행하여 여러분께서 생각한 결과와 일치하는지 판단해보세요. 만약 여러분이 생각한 결과와 일치하지 않는다면, 왜 일치하지 않는지 조사해보세요.

<Callout emoji="📌">
코드를 실행하기 전에 반드시 5분 이상 조사하고 고민하여 실행 결과를 예측해볼 수 있도록 하세요!
</Callout>

<br/>
## 1. 논리 연산자

[논리 연산자](../javascript-grammar/logical-operators.mdx)를 다시 한번 살펴보고 아래 퀴즈들을 풀어보세요.

<br/>
### 1-1. 퀴즈

```javascript copy showLineNumbers
const result = 30 && 0;

console.log(result);
```

<br/>
### 1-2. 퀴즈

```javascript copy showLineNumbers
const a = 0;
const result = !!a;

console.log(result);
```

<br/>
### 1-3. 퀴즈

```javascript copy showLineNumbers
const result = ("0" && 30) || false;

console.log(result);
```

## 2. 숫자

<br/>
### 2-1. 퀴즈

```javascript copy showLineNumbers
const result = 10 % 3;

console.log(result);
```

<br/>
### 2-2. 퀴즈

```javascript copy showLineNumbers
let a = 10 % 3;
a += 3;
const result = a++;

console.log(result);
```

<br/>
### 2-3. 퀴즈

```javascript copy showLineNumbers
let a = 10 % 3;
a -= 1;
const result = ++a;

console.log(result);
```

<br/>
### 3-3. 퀴즈

```javascript copy showLineNumbers
const today = "토요일";
const result = `나는 ${today}에 일한다!`;

console.log(result);
```

<br/>
### 3-4. 퀴즈

> 처음보는 방식이더라도, 한번 살펴보고 알아두세요.

- [문자열 toLowerCase 메서드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/toLowerCase)

```javascript copy showLineNumbers
const str = "HunGry?";
const result = str.toLowerCase();

console.log(result);
```

<br/>
### 3-5. 퀴즈

- [문자열 repeat 메서드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/repeat)
- [문자열 slice 메서드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/slice)

```javascript copy showLineNumbers
const str = "ha";
const a = str.repeat(5);
const result = a.slice(4, 6);

console.log(result);
```

<br/>
### 3-6. 퀴즈

- [문자열 split 메서드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/split)

```javascript copy showLineNumbers
const a = "ha".repeat(3);
const result = a.split("a");

console.log(result);
```
