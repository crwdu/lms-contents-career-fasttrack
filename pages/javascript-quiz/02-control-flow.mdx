# 반복문과 조건문 퀴즈

[반복문](../messenger-project-1/05-rendering#2-반복문.mdx)과 [조건문](../messenger-project-1/06-conditional-rendering#2-조건문.mdx)를 다시 한번 살펴보고 아래 퀴즈들을 풀어보세요.

<Callout emoji="📌">
코드를 실행하기 전에 반드시 5분 이상 조사하고 고민하여 실행 결과를 예측해볼 수 있도록 하세요!
</Callout>

<br/>
### 1. 퀴즈

```javascript copy showLineNumbers
let result = 0;

for (let i = 0; i < 5; i++) {
  if (i % 2 === 0) {
    result++;
  }
}

console.log(result);
```

<br/>
### 2. 퀴즈

```javascript copy showLineNumbers
let result = 0;

for (let i = 0; i < 10; i += 2) {
  if (i % 4 === 0) {
    result += i;
  }
}

console.log(result);
```

<br/>
### 3. 퀴즈

```javascript copy showLineNumbers
let result = 0;

for (let i = 0; i < 10; i++) {
  if (i % 3 === 0 || i % 4 === 0) {
    result += i;
  }
}

console.log(result);
```
