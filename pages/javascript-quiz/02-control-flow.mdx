# 반복문과 조건문 퀴즈

<Callout emoji="📌">
정확히 어떤 값이 콘솔에 출력되는지 3분 이상 잘 고민해보고, 아래 콘솔 출력문의 주석을 해제하여 정답을 확인해보세요.
