# 🖥️ 자바스크립트 연습장 프로그램

자바스크립트 코드를 간단하게 작성하고 실험해볼 수 있는 연습장 프로그램을 먼저 설치해보겠습니다.

아래 링크에서 다운로드 받아 실행시켜주세요.

🌏 [Run JS](https://runjs.app/)

다음과 같은 화면으로 프로그램이 실행됐나요? 😎

![runjs](/images/posts/runjs.png)

좌측 패널은 여러분이 코드를 입력하는 공간이고, 우측 패널은 여러분이 작성한 코드의 출력 결과를 확인할 수 있는 공간입니다.

좌측 공간에 다음과 같이 코드를 입력해보세요.

```javascript
console.log("hello world");
```

우측 공간에 다음과 같이 `'hello world'` 라는 내용이 표기된 것을 확인하셨나요?

![runjs](/images/posts/runjs-2.png)

정상적으로 출력 되었다면 모든 준비가 끝났네요. 아주 잘 하셨습니다. 💪🏻

이제 다음 단계에서 방금 작성한 자바스크립트 코드부터 이야기 해보도록 할까요?
