# 객체 자료형

마지막으로 객체 자료형은 가장 복잡한 구조의 자료형입니다. 객체 자료형에는 세부적으로 다음 3가지가 포함되어 있습니다.

1. 일반 객체
2. 배열
3. 함수

<br/>
## 1. 일반 객체

일반 객체는 [이전 단계](/messenger-project-1/04-data#2-자바스크립트-객체)에서 살펴봤던 key/value 형태의 자료를 담을 수 있는 구조의 자료형입니다.

```javascript copy
const person = {
  name: "ken",
  age: 10,
  address: "서울특별시"
};
```

<br/>
## 2. 배열

[배열](/carousel-project-1/13-javascript-11) 또한 객체 자료형에 속합니다. 앞으로 차차 살펴보겠지만, 배열은 객체의 하위 개념으로 객체의 모든 성질을 갖고 있으면서 동시에 배열만의 특징이 존재합니다.

```javascript copy
const list = [ "apple", "banana", "pineapple" ];
```

<br/>
## 2. 함수

마지막으로 객체 자료형에는 [함수](/carousel-project-1/11-javascript-9) 또한 포함됩니다.

```javascript copy
function add (a, b) {
  return a + b;
}
```

위 예시에서 `add`라는 이름의 함수는 함수임과 동시에 객체이기도 합니다.

<br/>
## Summary

자세한 내용은 앞으로 살펴보겠지만, 객체 자료형에는 일반 객체, 배열, 함수가 모두 포함된다는 것을 절대 잊지 마세요!
