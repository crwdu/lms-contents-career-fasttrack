# 코드 에디터 설치하기

프로젝트를 시작하기에 앞서, 우리가 코드 작업을 하기 위해 필요한 프로그램을 먼저 설치해야 합니다.

그 중 가장 필수적인 프로그램은 코드 에디터인데요. 코드 에디터란, 코드를 작성하기 위한 메모장 같은 역할을 하는 프로그램이라고 생각할 수 있습니다. 다만, 코드를 작성하는데 편리한 여러 가지 편의성 기능들이 추가되어 있습니다.

요즘 가장 대중적으로 많이 사용하는 Visual Studio Code를 설치해주세요.

<Callout emoji="📂">
  [다운로드 받기](https://code.visualstudio.com/download)
</Callout>

설치가 완료되었다면, 이제 코드를 작성할 준비가 모두 되었습니다. 💫
