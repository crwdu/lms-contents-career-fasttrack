# 🖥 HTML 작업 시작하기

자, 이제 실전으로 들어가 보겠습니다. 👊🏻

방금 만들 여러분의 `index.html` 파일에 아래와 똑같이 내용을 추가해보세요.

```html copy showLineNumbers filename="index.html"
<html lang="ko">
  <head>
    <!-- 모르는 태그가 있더라도 아직 신경쓰지 마세요 🥸 -->
    <title>🔰 바닐라코딩</title>
  </head>
  <body>
    <h1>바닐라코딩</h1>
  </body>
</html>
```

## 페이지 제목 넣기

잠시 돌아가, 우리의 `index.html` 파일의 내용을 다시 한번 살펴보도록 하겠습니다.

```html copy showLineNumbers filename="index.html"
<html lang="ko">
  <head>
    <!-- 모르는 태그가 있더라도 아직 신경쓰지 마세요 🥸 -->
    <title>🔰 바닐라코딩</title>
  </head>
  <body>
    <h1>바닐라코딩</h1>
  </body>
</html>
```

우리가 작성한 HTML 문서의 `head` 태그 내부에 보면, `title` 이라는 태그가 사용된 것을 확인할 수 있습니다. `title` 태그는 브라우저 윈도우 탭에 표기되는 웹사이트 타이틀을 설정할 수 있도록 합니다.

그렇기에, `title` 태그 내부에 컨텐츠로 작성된 `🔰 바닐라코딩` 이라는 텍스트가 우리 웹사이트의 탭에 제목으로 표기됩니다.

쉽죠? 😌

🌏 [MDN HTML title tag](https://developer.mozilla.org/ko/docs/Web/HTML/Element/title)

## 코멘트 작성하기

우리가 작성한 HTML 문서에는 또 신기한 부분이 있습니다.

```html
<!-- 모르는 태그가 있더라도 아직 신경쓰지 마세요 🥸 -->
```

위와 같이 작성한 부분인데요. `head` 태그 내부에 작성되었기에 화면에 표기되지 않는 것은 당연하다 하더라도, 과연 어떤 기능을 하는 코드일까요?

이런 부분을 우리는 코드에서 코멘트(혹은 주석)이라고 말하는데요. 코드의 기능에 영향을 끼치지 않고 메모를 남기는 것 같은 역할을 합니다. 그래서 위의 저 문장은 제가 여러분에게 남긴 메모랍니다. 😎

모든 프로그래밍 언어에는 코멘트를 작성할 수 있는 문법적 기능이 있습니다. 우선 현재 우리가 살펴보고 있는 HTML에서 코멘트는 아래와 같이 작성합니다.

![html comment](/images/posts/html-intro/comment.png)
