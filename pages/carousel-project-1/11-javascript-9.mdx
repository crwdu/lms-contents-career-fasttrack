# 코드의 집합, 함수

자바스크립트에서 함수는 **여러 개의 명령어를 묶음 단위로 편리하게 실행할 수 있는 기능**을 제공합니다.

<br/>
## 1. 함수 만들기

함수는 아래와 같이 만들 수 있습니다.

![functions-1](/images/posts/functions-1.png)

코드로 작성해보자면,

```javascript copy showLineNumbers
function greet() {
  const myName = "ken";
  console.log(`Hello ${myName}`);
}
```

위의 예시 코드에 대해 기억해야 할 부분이 있습니다.

1. `greet` 이라는 이름의 새로운 변수를 만드는 것과 같이, 자바스크립트는 `greet` 이라는 이름을 이해할 수 있게 됩니다.
2. 위의 예시 코드는 `greet` 이라는 함수를 **만드는** 코드입니다. 함수를 만들었다는 의미는 함수 내부의 코드를 실행하는 것과는 다른 뜻입니다. 그렇기에, 함수 내부의 코드가 실행되지는 않습니다.

다음과 같은 코드를 한번 실행시켜보세요.

## 2. 함수 실행하기

함수 내부에 작성된 코드를 실행하려면 함수를 호출해야만 합니다. 함수는 "호출한다", "실행한다" 등의 용어를 사용합니다.

함수를 호출하려면 다음과 같이 **함수 이름 뒤에 소괄호**를 붙여 호출할 수 있습니다.

![functions-2](/images/posts/functions-2.png)

코드로 작성해본다면 아래와 같습니다.

```javascript copy showLineNumbers /greet();/
function greet() {
  const myName = "ken";
  console.log(`Hello ${myName}`);
}

greet(); // greet 함수 호출
```

위와 같이 `greet` 함수를 호출하는 코드를 추가하면 함수 내부 코드가 실행됩니다.

아래에서 코드가 실행되는 결과를 직접 확인해보세요.

## 3. 함수의 실행결과, 반환값

함수는 **반드시** 호출에 대한 결과를 갖습니다. 함수 호출에 대한 결과를 우리는 **반환값**이라고 합니다.

**함수의 반환값이라는 것은 `return` 이라는 문법 키워드를 이용하여 지정할 수 있고, `return` 키워드 뒤에 작성한 내용이 반환됩니다. 💁🏻‍♂️**

```javascript copy showLineNumbers /return/
function getGreeting() {
  const myName = "ken";
  return `Hello ${myName}`; // "Hello ken" 반환
}
```

**함수의 반환값은 함수 호출 위치에 함수 호출문을 대체하여 들어가는 것과 같습니다.**

위에서 봤던 예시 코드를 다시 조금 변경하여 다시 살펴볼까요?

```javascript copy showLineNumbers /getGreeting();/
function getGreeting() {
  const myName = "ken";
  return `Hello ${myName}`;
}

const result = getGreeting();
console.log(result);
```

위와 같이 코드를 작성한 경우, 6번째 줄에서 `getGreeting` 함수가 호출되어 `"Hello ken"` 이라는 문자열이 반환(return)됩니다. 그리고 이 함수의 반환값인 `"Hello ken"`이 `result` 변수에 대입되어 7번째 줄에서 출력되고 있습니다.

![functions-5](/images/posts/functions-5.png)

아래의 코드 에디터에 직접 출력 결과를 확인해보고, 작성된 코드에서 3번째 줄 `return` 뒤에 작성된 구문을 여러분이 원하는 문자열이나 숫자로 변경하고 실행해보세요.

## 4. 반환값을 지정하지 않았을때

함수에서 명시적으로 `return` 을 사용하지 않아도 함수의 반환값은 **반드시** 존재합니다.

```javascript copy showLineNumbers
function logGreeting() {
  const myName = "ken";
  console.log(`Hello ${myName}`);
}

const result = logGreeting();
console.log(result);
```

함수의 반환값은 `return` 문이 결정합니다. 위의 예시 코드에 작성된 함수 내부에는 `return` 이 사용되지 않았습니다.

위의 예시 함수처럼 `return` 이 명시적으로 되지 않은 경우, 함수의 반환값은 기본적으로 **undefined**라는 값입니다.

<Callout emoji="📌">
  `undefined`에 대해서는 다음에 다시 자세히 알아보도록 하겠습니다.
</Callout>

## 5. 함수의 매개변수

함수는 외부에서 데이터를 전달받아 함수 내부 연산에 사용할 수 있는데, 이를 매개변수라고 합니다.

함수를 만들때 아래와 같이 매개변수에 대한 가정을 하고 함수 내용을 작성합니다.

```javascript copy showLineNumbers
function getGreeting(name) {
  return `Hello ${name}`;
}
```

함수 내용은 함수가 호출되어야 실행됩니다. 매개변수 또한, 함수를 만들 당시에는 이 자리에 어떤 값이 들어올 예정이라는 표식만 해두는 것이며, 실제로 어떤 값이 들어오는지는 함수가 호출되는 시점에 결정됩니다.

```javascript copy showLineNumbers
function getGreeting(name) {
  return `Hello ${name}`;
}

const result = getGreeting("ken");
console.log(result);
```

위와 같이 5번째 줄에서 `getGreeting` 함수를 호출할 경우, 호출 시점에 `name` 이라는 매개변수에 `"ken"` 이라는 문자열이 대입되어 `getGreeting` 함수가 실행됩니다.

매개변수를 사용하면, 하나의 함수를 활용하여 상황에 맞춰 동적으로 대응할 수 있습니다.

```javascript copy showLineNumbers
function getGreeting(name) {
  return `Hello ${name}`;
}

const greetingToKen = getGreeting("ken");
const greetingToWan = getGreeting("wan");

console.log(greetingToKen);
console.log(greetingToWan);
```

아래 에디터에서 직접 실행해보고, `"ken"`이나 `"wan"`을 여러분이 원하는 문자열로 변경해보세요.

## 6. 함수의 종료

`return` 이라는 문법 키워드는 그 특성상 함수 내부에서만 사용할 수 있는데 또 한가지 중요한 기능이 있습니다.

**`return` 이 실행되는 순간, 함수 내부 코드는 남은 코드 내용을 실행하지 않고 즉시 실행이 종료됩니다.**

코드로 보는게 조금 더 이해가 쉬울것 같으니, 예시를 볼까요? 🥸

> 처음보는 문법이라도 대략적인 흐름만 이해해보세요.

<br/>
<Sandpack
  template="vanilla"
  theme={nightOwl}
  files={{

      code: `function eatDinner(menu) {
  if (menu === "burger") {
    console.log("조건문 내부");
    return "good";
  }

  console.log("조건문 아랫 부분");
  return "no good";
}

const feeling = eatDinner("burger");
console.log(feeling);`,
      active: true

  }}
  options={{

  }}

/>

위와 같이 코드를 실행하는 경우, 출력문은 1개만 실행됩니다.

우리가 작성한 내용을 보면 `menu`가 `"burger"`와 일치할 경우, `"조건문 내부"`라는 문자열을 출력하고 `return` 문은 함수의 실행을 즉시 종료시키기 때문이죠. 💁🏻‍♂️

함수는 우리가 코드를 작성할때 매우 기본적인 단위로 자주 사용하는 묶음이기에 기본적인 기능에 대해 정확히 이해하고 사용해야만 합니다.

- 함수를 만들고 호출되는 것
- 함수의 반환값
- 함수의 매개변수
