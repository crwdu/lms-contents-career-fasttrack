# 순서대로 정리하는 배열

배열은 흔히 아래와 같이 대괄호로 표기합니다.

```js
const fruits = ["apple", "orange", "grape"];
```

<br/>
## 1. 배열의 요소

배열 안에 콤마(`,`)로 구분되어 담겨있는 값들을 **요소**라고 합니다.

```js
const fruits = ["apple", "orange", "grape"];
```

위의 `fruits` 변수에 담긴 배열의 경우,

1. 첫 번째 요소는 `'apple'`,
2. 두 번째 요소는 `'orange'`,
3. 세 번째 요소는 `'grape'`입니다.

그래서 `fruits` 배열은 3개의 요소를 갖고 있다고 말할 수 있습니다.

<br/>
## 2. 배열의 길이

모든 배열은 요소의 갯수에 대한 정보를 아래와 같이 `.length` 속성을 제공합니다. (문자열과 동일합니다.)

```javascript copy
const fruits = ["apple", "orange", "grape"];

console.log(`과일은 ${fruits.length}가지가 있습니다.`);
```

## 3. 배열과 함수 함께 사용해보기

이전에 함수에 대하여 배웠으니, 함수를 응용하여 코드를 한번 작성해볼까요?

아래에 작성된 `getNumOfFruits` 함수는 `fruits` 배열의 길이를 사용하여 문자열을 콘솔에 출력합니다.

```javascript copy showLineNumbers
const fruits = ["apple", "orange", "grape"];

function getNumOfFruits() {
  console.log(`과일은 ${fruits.length}가지가 있습니다.`);
}

getNumOfFruits();
```

## 4. 요소의 순번, 인덱스

배열의 요소들은 각각 정해진 순번이 있는데, 이 순번을 인덱스(index)라고 부릅니다.

인덱스는 특이하게도, 0부터 시작하는 체계로 이루어져 있습니다.

예를 들어,

```js copy showLineNumbers
const fruits = ["apple", "orange", "grape"];
```

위와 같은 배열이 있다고 한다면, `fruits` 배열의 0번 인덱스에는 `"apple"`이 있고, 1번 인덱스에는 `"orange"`가 있고, `"grape"`는 2번 인덱스에 위치해 있습니다.

배열의 특정 인덱스에 접근하기 위해서는 대괄호 문법을 활용해야 합니다.

```js copy showLineNumbers
const fruits = ["apple", "orange", "grape"];

console.log(`${fruits[0]}은 사과입니다.`);
console.log(`${fruits[1]}은 오렌지입니다.`);
console.log(`${fruits[2]}은 포도입니다.`);
```

위에 주어진 코드에서,

- `fruits[0]`은 `"apple"` 문자열을 가리키고,
- `fruits[1]`은 `"orange"` 문자열을 가리키고,
- `fruits[2]`은 `"grape"` 문자열을 가리키게 됩니다.

그리고 문자열은 [문자열의 길이를 판별할 수 있는 방법](../carousel-project-1/09-javascript-7.mdx#3-문자열의-길이)이 있습니다.

이 두 가지를 조합하여 생각해본다면, 우리는 주어진 배열의 각 요소의 길이를 판별할 수도 있습니다.

```js copy showLineNumbers
const fruits = ["apple", "orange", "grape"];

console.log(`${fruits[0]}은 사과이고, ${fruits[0].length}글자로 이루어져 있습니다.`);
console.log(`${fruits[1]}은 오렌지이고, ${fruits[1].length}글자로 이루어져 있습니다.`);
console.log(`${fruits[2]}은 포도이고, ${fruits[2].length}글자로 이루어져 있습니다.`);
```

## 5. 요소 변경하기

인덱스를 활용하여 배열의 특정 요소를 추가하거나 변경할 수도 있습니다.

아래에 주어진 코드를 보면, 5번째 줄에서 `fruits` 배열의 0번째 인덱스에 위치에 `"kiwi"`라는 문자열을 대입하고 있습니다. 이와 같은 코드가 실행되게 되면, 0번째 인덱스 위치에 `"kiwi"` 문자열이 삽입되게 됩니다.

<Callout emoji="📌">
  기존에 0번째 위치에 있던 요소는 사라집니다.
</Callout>

```js copy showLineNumbers {5}
const fruits = ["apple", "orange", "grape"];

fruits[0] = "kiwi";
```

아래 에디터에서 실행 결과를 직접 확인해보고, 0으로 되어 있는 인덱스를 다른 인덱스 위치로 변경해보세요.
