# Props

## 정의

_Props란, **함수에서의 매개변수**와 동일한 개념입니다._ **즉, 외부에서 전달받는 인풋이라는 뜻입니다.**

함수의 매개변수를 선언하는 용도, 함수의 매개변수를 네이밍 하는 방법 등을 반드시 명심하세요.

React Component에게 Attribute을 주게 되면, 해당 컴포넌트 내부에서는 그것들을 `props`이라고 부릅니다. 그리고 우리는 우리가 원하는 만큼 다양하게 `props`를 줄 수 있습니다.

```jsx showLineNumbers {2, 5, 13}
function Article (props) {
  const { prop1, prop2, prop3 } = props;

  return (
    <p>{prop1}, {prop2}, {prop3}</p>
  );
}

function App () {
  return (
    <>
      <h1>Vaco App</h1>
      <Article prop1={1} prop2={2} prop3={3} />
    </>
  );
}
```

## 이벤트 등록하기

리액트 컴포넌트에서는 `on` + 이벤트 이름(Camel case)로 이벤트 등록을 처리합니다.

`onClick`, `onMouseOver`, `onKeyPress` 등.

> `on` + 이벤트 이름 대문자로 시작 + 이벤트 이름 카멜 케이스

```jsx showLineNumbers {9}
function App () {
  function handleListClick (ev) {
    alert(ev.target.textContent);
  }

  return (
    <div>
      <h1>My list</h1>
      <ul onClick={handleListClick}>
        <li>List 1</li>
        <li>List 2</li>
        <li>List 3</li>
      </ul>
    </div>
  );
}
```

리액트는 React Element가 요소로 담긴 배열을 렌더링할 경우, 화면에 해당 요소들을 자동으로 렌더링 합니다. 방금 위 예시에서도, 배열 내에 있는 `TodoItem` 세개가 화면에 표기되었습니다.

만약 Todo list 데이터가 서버에서 전달된다고 가정하고 리팩터링 해본다면, 다음과 유사할 수 있습니다.

<br />

<Sandpack
  template="react"
  theme={githubLight}
  files={{
    "/App.js": {
      code: `const list = [
  {
    id: 1,
    text: "돈까스 먹기"
  },
  {
    id: 2,
    text: "대청소 하기"
  },
  {
    id: 3,
    text: "넷플릭스 보기"
  }
];

function TodoItem ({ number, todo }) {
  return (
    <p>#{number}: {todo}.</p>
  );
}

export default function TodoList () {
  return (
    <div className="something">
      <p style={{ fontSize: "30px" }}>
        오늘의 할일
      </p>
      {list.map(function (item) {
        return (
          <TodoItem key={item.id} number={item.id} todo={item.text} />
        );
      })}
    </div>
  );
};
`,
      active: true
    }
  }}
  options={{
    showLineNumbers: true,
    editorHeight: "760px",
    editorWidthPercentage: 60,
    initMode: "lazy",
    autoReload: false,
    showConsoleButton: true,
  }}
  customSetup={{
    entry: "index.js",
  }}
/>

> 위 예시에서 `key`라는 prop은 무슨 용도일까요?

> `Array.prototype.map` 함수의 반환값은 무엇인지 잘 상기시키고 예시 코드를 이해하셔야 합니다.
