## `Array.prototype.reduce`

<br />

배열의 `reduce` 함수에 대해 기억하시나요?

우선, `reduce` 함수는 주어진 배열(현재, `list`)를 순회하며 주어진 작업을 합니다. 구체적으로 주어진 배열에서 최대값을 구하는 아래 예시의 흐름을 한번 살펴봅시다.

```js showLineNumbers
const list = [ 3, 2, 5, 10, 7 ];

const max = list.reduce(function reducer (currentMax, value) {
  return currentMax < value ? value : currentMax;
});
```

`reduce` 함수는 인수로 함수를 받을 수 있고 이 함수를 우리는 `reducer`라고 부릅니다. `reducer` 함수는 아래와 같이 호출됩니다.

|              | currentMax        | value              |  반환값  |
|--------------| -----------       | ------------       |  ------|
| 첫번째 호출     | 3 (배열의 첫번째 요소) | 2 (배열의 두번째 요소) |  3     |
| 두번째 호출     | 3 (이전 호출의 반환값) | 5 (배열의 세번째 요소) | 5      |
| 세번째 호출     | 5 (이전 호출의 반환값) | 10 (배열의 네번째 요소) | 10    |
| 네번째 호출     | 10 (이전 호출의 반환값)| 7 (배열의 다섯번째 요소) | 10 (최종 결과값)   |

이와 같이, `reduce`는 `reducer` 함수를 이용해, **이전 결과와 현재 요소를 활용해 다음 결과를 연산**하도록 합니다.

<br />
<br />

> 알고 계시겠지만 `reduce` 함수는 초기값에 대한 설정 또한 가능합니다. 초기값 설정을 할 경우, `reducer` 함수가 최초 호출되는 시점에 초기값과 함께 배열의 첫번째 요소가 인수로 주어집니다. 자세한 내용은 MDN의 [`Array.prototype.reduce`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/reduce) 페이지의 Example 파트를 참고해주세요.

<br />
<br />

여러분께서 `reduce`의 로직을 잘 이해하셨다면, 아래 예시 코드가 어떻게 동작하는지 위에서 보여드린 테이블처럼 한번 정리해보시기 바랍니다.

```js showLineNumbers
function addThree (x) {
  return x + 3;
}

function multiplyFive (x) {
  return x * 5;
}

function divideByTwo (x) {
  return x / 2;
}

const operations = [ divideByTwo, multiplyFive, addThree ];

const result = operations.reduce(function reducer (input, func) {
  return func(input);
}, 10);
```
