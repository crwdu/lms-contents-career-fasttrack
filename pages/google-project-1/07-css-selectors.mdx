# 더 많은 CSS 선택자

추가적으로 CSS 선택자 문법을 어떻게 더 활용해볼 수 있는지 살펴보도록 하겠습니다.

<br/>
## 1. Type

```css
h1 {
  color: blue;
}
```

HTML의 태그 종류(type)을 지정하여 요소들을 꾸밀 수 있습니다. 지정한 태그 종류에 해당하는 _모든_ 요소들은 설정한 스타일이 적용됩니다.

<br/>
### 1-1. 문서의 모든 `h1` 태그에 적용되는 스타일

```css /h1/
h1 {
  color: blue;
}
```

<br/>
### 1-2. 문서의 모든 `h2` 태그에 적용되는 스타일

```css /h2/
h2 {
  color: red;
}
```

<br/>
### 1-3. 문서의 모든 `ul` 태그에 적용되는 스타일

```css /ul/
ul {
  padding: 0;
}
```

이와 같이, 여러분이 어떤 태그의 종류를 선택하여 공통의 스타일을 적용하고 싶다면 위와 같이 태그의 종류를 명시하여 스타일을 부여할 수 있습니다.

아래 에디터에서 `styles.css`를 선택하여 여러분이 원하는 스타일을 한번 추가해보세요.

## 2. Class

HTML 요소에게는 클래스라는 속성을 부여하여 식별할 수 있습니다. 특정 요소에게 주어진 클래스를 이용하여 스타일을 명시할 수 있습니다.

아래와 같은 HTML 문서가 있다면, `p` 요소에게 `thick`이라는 클래스가 주어져 있습니다.

```html
<p class="thick">I'm thick</p>
```

앞서 살펴본 것처럼 태그의 종류(`p`)을 활용하여 선택하고 스타일을 부여할 수도 있지만, 그럴 경우 문서 내의 모든 `p` 태그에게 스타일이 적용됩니다.

그렇기에, 클래스는 우리가 원하는 스타일을 조금 더 선택적으로 적용할 수 있는 수단을 제공합니다.

```css
.thick {
  font-weight: bold;
}
```

<Callout emoji="📌">
  CSS 파일에서 클래스를 이용하여 선택할 때는 클래스 이름 앞에 `.`을 적어주어야 합니다.
</Callout>

### 2-2. 여러 개의 클래스

또한 우리는 하나의 HTML 요소에게 하나 이상의 클래스를 부여할 수 있습니다.

```html
<p class="thick alert">Warning...</p>
```

여러 개의 클래스를 하나의 HTML 요소에게 부여할 경우에는 위와 같이 공백을 기준으로 나누어 표기해주도록 합니다. 부여할 수 있는 클래스의 갯수에는 제한이 없습니다.

아래와 같이 CSS를 작성했다고 생각해보세요. 어떤 결과가 나타날까요?

```css
.thick {
  font-weight: bold;
}

.alert {
  color: red;
}
```

우리가 작성한 `p` 태그에게는 `color: red;`와 `font-weight: bold;`가 모두 적용됩니다.

<Callout emoji="📌">

## 3. ID

HTML 문서에서 우리는 각 요소에게 아래와 같이 `id`라는 속성을 부여할 수 있습니다.

```html
<div id="blue-box">I'm a blue box</div>
```

`id`는 HTML 문서 내에서 유일해야 하기 때문에 위와 같은 코드를 HTML에 작성했다면, 해당 HTML 문서 내에서 `box`라는 `id`를 가진 요소는 더 이상 없어야 합니다.

<Callout emoji="📌">
**HTML 요소의 id란?**

HTML 요소에는 class와 유사하게, id라는 속성을 부여할 수 있습니다.

하나의 HTML 문서에서 여러 개의 요소들이 같은 class 이름을 사용하는 것이 허락되는 반면, **id는 반드시 고유해야 합니다.**
</Callout>

```css
#blue-box {
  background-color: blue;
  color: white;
}
```

CSS에서 HTML 요소의 `id`를 이용하여 선택할 때는 `id` 이름 앞에 `#` 기호를 붙여주어야 합니다.

## 5. Descendent

```css
#nav li {
  background-color: blue;
}
```

이 선택 방식은 특정 요소의 하위 요소를 선택할 때 사용됩니다. 두 개의 선택자 사이에 공백을 한 칸 주게 되면, 공백을 기준으로 왼쪽에 명시된 선택자의 하위에 존재하며 왼쪽에 명시된 선택자를 가진 HTML 요소를 선택하게 됩니다. 위 예제에서는 `nav`라는 ID 값을 가진 요소의 하위에 존재하는 모든 `li` 태그 요소들을 선택하여 스타일을 명시하고 있습니다.

```html
<ul id="nav">
  <li>child</li>
</ul>
```

## 6. Child

```css
#list > li {
  border: 1px solid black;
}
```

부등호 기호를 위와 같이 사용하여 직속 자식 요소들을 선택할 수도 있습니다. 위 예제의 경우, `list`라는 ID 값을 가진 요소의 직속 자식 요소들 중 `li` 태그 요소들만을 선택하여 스타일을 적용하게 됩니다. 바로 하위가 아닌 요소들은 제외됩니다.

```html
<ul id="list">
  <li>child</li>
  <li>
    <p>child 2</p>
    <ul>
      <li>grand child 1</li>
      <li>grand child 2</li>
      <li>grand child 3</li>
    </ul>
  </li>
</ul>
```

## 8. Attribute

```css
img[alt="cat"] {
  border: 5px solid red;
}
```

위 선택자는 `img` 태그들 중 `alt` 속성의 값이 `"cat"`인 HTML 요소들을 선택하여 스타일을 적용합니다. `alt` 이외의 다른 속성들을 이용하여 선택할 수도 있습니다.

