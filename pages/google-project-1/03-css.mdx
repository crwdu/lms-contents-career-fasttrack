# 📗 웹을 꾸미는 디자이너, CSS

CSS는 **Cascading Style Sheets**의 약자입니다.

> **📖 Cascading**
>
> 폭포가 되어 떨어지다

CSS는 기본적으로 상위 요소에 적용한 스타일 내용이 하위 요소에게 상속되는 구조를 갖습니다. 그렇기에 이름에서도 Cascading이라는 단어가 사용되는 것입니다.

CSS는 웹사이트를 꾸미는 목적으로 사용됩니다.

* 특정 요소에 색깔을 입히거나,
* 특정 요소의 위치를 조정하거나,
* 특정 요소의 크기나 굵기 등을 조절하거나,

이와 같이, 웹사이트를 꾸미는 목적으로 CSS는 매우 다양한 기능을 제공합니다.

다시 정리하자면, **HTML은 웹 문서의 컨텐츠와 구조를 결정하고, CSS는 컨텐츠가 어떤 식으로 꾸며져야 하는지를 결정합니다.**
