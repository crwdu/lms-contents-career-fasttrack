# 🖥 Google 프로젝트

우리는 이제 프로젝트를 만들어 보며, 조금 더 구체적인 HTML의 개념과 CSS에 대해서 살펴보도록 하겠습니다.

![google](/images/posts/html-intro/google.png)

구글의 홈페이지를 함께 만들어볼텐데요.

<br />
## 1. 주요 개념

- HTML의 적용
- 시맨틱 마크업 (Semantic Markup)
- Block, Inline, Inline-Block 요소의 차이점
- CSS 문법과 적용
- CSS 선택자
- 웹 페이지 마크업 구조 설계 방법
- CSS로 가운데 정렬하는 방법
- CSS 포지션을 활용한 레이아웃 설계 방법
- CSS 박스 모델
- CSS 플렉스 박스

<br />
## 2. 학습 목표

위에 나열된 개념들을 이용하여 구글 홈페이지와 최대한 똑같이 생긴 웹 페이지를 작업해 보는 것이 목표입니다.

이 가이드를 따라 하시면 50% 정도가 완성되고, 나머지 50%는 여러분 스스로 완성해보며 이해한 내용을 적용해보는 시간이 필요합니다.
