# Inline, Block, Inline-Block

Inline, Block, Inline-Block은 하나의 HTML 요소가 화면에 배치될때 어떤 식으로 배치되는지에 대한 속성입니다.

모든 HTML 요소들은 별도의 설정이 없더라도 기본적으로 Inline, Block, 혹은 Inline-Block 세 가지 중 하나의 속성을 지니고 있습니다. 또한 우리가 원한다면, 필요에 의해 변경할 수 있습니다.

우선 간단하게 설명하자면, **Block 요소는 화면의 세로 방향으로 나열되는 박스 형태의 요소를 뜻하고, Inline 요소는 가로 방향으로 나열되는 요소**입니다.

그리고 위 두 가지의 성질을 복합적으로 지닌 것이 Inline-Block 요소입니다.

이것들의 차이점을 명확히 알고 있는 것은 굉장히 중요하며, 여러 상황에서 유용하게 응용될 수 있습니다.

그렇다면 각각의 속성에 대해 조금 더 자세히 알아볼까요?

<br/>
## 1. Block Elements

Block 요소들은 화면의 세로선을 축으로 생각했을때, 항상 새로운 줄에서 시작하고 위 아래에 서로 쌓이는 형식으로 나타납니다. 그리고 가로선 상으로는 화면의 사용 가능한 최대 너비를 차지하게 됩니다.

쉽게 말하자면, 우리가 보는 화면의 **세로선 상에서 위에서 아래의 방향으로 나열**됩니다.

아래의 예제를 한번 보시기 바랍니다. HTML 상으로는 한 줄에 2개의 `p` 태그를 작성하였습니다. 하지만 화면에서 보시면, 두 개의 `p` 태그가 모두 전체 화면의 너비만큼 자리잡은 상태로 각각 새로운 세로선 상에 나타나는 것을 볼 수 있습니다.

`p` 태그는 브라우저 상에서 기본으로 Block 요소이기 때문에, 항상 새로운 줄에서 요소가 시작하고, 가로 상으로는 최대 너비를 차지하도록 배치됩니다.

## 2. Inline Elements

Inline 요소들은 새 줄에서 나타나지 않습니다. 일반적인 문서에서의 텍스트 흐름과 동일하게 **가로선 상에서 좌측에서 우측으로 나열**됩니다.

각각의 Inline 요소는 포함하고 있는 콘텐츠 영역 만큼의 너비만을 차지하고, 하나에 이어 또 다른 하나가 가로선 상에 이어 붙어서 나타나는 형식입니다.

`span` 태그는 대표적인 Inline 요소 중 하나입니다.

화면 상으로 초록색 배경이 적용된 부분들이 보이시나요? 각각이 다른 Inline 요소들입니다.

새로운 줄로 넘어가지 않고 텍스트의 흐름 상에 자연스럽게 적용됩니다.

**노란 배경을 가진 `p` 태그는 Block 요소이기 때문에 가로선 상에서 최대치의 너비를 차지하고, 그 내부의 콘텐츠는 각각의 콘텐츠 너비만큼의 크기를 차지하게 됩니다.**

<br/>
### 2-1. 가운데 정렬하기

Inline 요소를 가로선 상에서 가운데 정렬하기 위해서는, 가운데로 정렬시키고 싶은 Inline 요소의 부모 요소에게 `text-align: center;`를 적용해주면 됩니다.

이전에 Block 요소에 대해 살펴봤던 예시를 활용해보도록 하겠습니다.

`p` 태그에게 `text-align: center;`가 주어져 있기 때문에, 하위에 자리잡은 모든 Inline 요소가 가로선 상에서 가운데 정렬하였습니다.

> 일반 텍스트 또한 Inline 요소입니다.

## 3. Inline Block Elements

위에서 언급한 것처럼 Inline 요소에게 Block 요소와 같은 속성을 사용하고 싶을때, 혹은 반대로 Block 요소에게 Inline 요소와 같은 속성을 사용하고 싶을때 사용할 수 있는 것이 바로 Inline Block입니다.

Inline Block은 이름처럼, Inline과 Block의 성질을 모두 지니고 있습니다.

<br/>
### 3-1. 여러 개의 Block 요소를 한 줄에 배치하기

`inline-block`을 이용하면, 여러 개의 Block 요소들을 마치 Inline 요소처럼 한 줄에 이어서 배치할 수 있습니다.

아래 예제를 보시면 이전 예제와 약간의 스타일 차이는 있지만, `<section>` (Block) 요소와 `<aside>` (Block) 요소에 `display: inline-block;`을 주어 두 개의 Block 요소를 한 줄에 나란히 배치한 것을 볼 수 있습니다.

Inline Block 요소는 Inline 요소처럼 **콘텐츠 너비만큼만의 가로 공간을 차지**하게 되어 한 줄에 여러 개의 요소를 나열할 수 있습니다.

<br/>

> Inline Block 요소에게 높이를 지정해줄 경우, 세로선 상에서 위치가 어긋날 수 있습니다.
>
> `vertical-align: middle;` 속성을 활용하면, Inline Block 요소들을 세로선 상에서 같은 위치에 배치할 수 있습니다.

앞으로 작업할 프로젝트에서 다양한 상황에 적용해보며 활용법을 더욱 익혀보도록 할테니, 지금은 우선 예시 코드의 내용들을 잘 이해해보도록 하세요. 🙇🏻

<br/>
## 4. 중첩 구조

Block 요소와 Inline 요소가 중첩되는 구조에서 지켜야 할 몇 가지 규칙이 있습니다. 반드시 지켜야만 하는 것은 아니지만, 특별한 상황이 아니라면 항상 지키는 규칙입니다.

- Inline 요소 내부에는 Block 요소를 배치하지 않습니다.
- Inline 요소 내부에는 Inline 혹은 Inline Block 형태의 요소를 배치합니다.
- Block 요소 내부에는 Block, Inline, Inline Block 모두 배치할 수 있습니다.

요약하자면, Inline 요소 내부에는 Block 요소를 최대한 배치하지 않는 편이라는 것을 잘 기억하시면 좋을것 같습니다.

> 몇 가지 예외사항은 있습니다만, 앞으로 다양한 경험을 쌓으며 배워나가실테니 지금은 저 규칙을 지키며 작업해보세요.
